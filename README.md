# DCS 88
### Run dev
MySQL server must be running and with info in .env
1. Navigate to backend
```bash
cd backend
```
2. Install dependencies
```bash
npm i
```
3. Start server
```bash
node index.js
```
OR
```bash
nodemon index.js
```
4. Navigate to frontend
```bash
cd ../frontend
```
5. Install dependencies
```bash
npm i
```
3. Start dev vite
```bash
npm run dev
```
7. Open browser for given url
```bash
http://localhost:5173
```
